# CPI-and-Money-Supply
Bank of Canada Inflation and Money Supply Dashboard

This application dynamically fetches data from the Bank of Canada Valet API to display interactive charts comparing Total CPI, CPI-Trim, CPI-Median, and CPI-Common against M1+, M1++, and M2++ money supply metrics.

## Development
To run this project locally:
1. Ensure you have Node.js installed.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the local Vite development server.
